# Light-Novel-JLPT
Read Japanese light novels and have Japanese Language Proficiency Test (JLPT).

To scrape corpus from [](https://yomou.syosetu.com):

`$ python3 scraper.py`

To clean and parse corpus:

`$ python3 parser.py`

To build and save N-gram language model:

`$ python3 build.py `

To test questions:

`$ python3 test.py`
